# FlappyBird-Python
This project is a game of FlappyBird that plays itself using NEAT in Python.

# Requirements
 - Python 3.x
 - NEAT
 - PyGame
 
 # Instructions To Use
   - **Make sure all files should be in the same folder before running.**
   ``` 
   pip install -r requirements.txt
   ```
   ```
   python3 FlappyBird.py
   ```
